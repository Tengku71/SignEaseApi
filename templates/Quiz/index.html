Aplikasi Pengenal Bahasa Isyarat

Deteksi bahasa isyarat secara real-time menggunakan AI

Beranda Mode Quiz
Arahkan tangan Anda ke kamera
Petunjuk Penggunaan
  1. Pastikan tangan Anda terlihat jelas dalam bingkai kamera.
  2. Lakukan gerakan bahasa isyarat yang ingin dideteksi.
  3. Hasil deteksi akan muncul secara otomatis.
  4. Gunakan mode Quiz untuk berlatih dan menguji kemampuan Anda.
Tentang Aplikasi

Aplikasi ini menggunakan teknologi pembelajaran mesin dengan MediaPipe untuk mendeteksi gerakan tangan dan model Multilayer Perceptron (MLP) untuk mengklasifikasikan bahasa isyarat.

Aplikasi dapat mengenali huruf A sampai X dalam bahasa isyarat Indonesia.
